📋 Setup Guide

วิธี deploy บน GitHub Pages
  1. สร้าง GitHub repo ใหม่ (public หรือ private + Pages enabled)
  2. Push ไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repo
  3. ไปที่ Settings → Pages → Source: main branch / root
  4. เพิ่ม Secrets (Settings → Secrets → Actions):
    • TABLEAU_USERNAME = email เข้า bi.learn.co.th
    • TABLEAU_PASSWORD = password
  5. GitHub Actions จะ sync ข้อมูลทุก 30 นาทีโดยอัตโนมัติ
  6. URL ของเว็บจะเป็น https://[username].github.io/[repo-name]/
Manual trigger: GitHub → Actions tab → "Sync Tableau Data" → Run workflow
